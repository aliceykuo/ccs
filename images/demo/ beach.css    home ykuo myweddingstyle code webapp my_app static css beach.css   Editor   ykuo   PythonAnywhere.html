Creating a virtualenv

PythonAnywhere has many useful modules pre-installed, but they may not be the same version as the ones you need for your web app. Check the batteries included page for details of exactly what we have installed for each Python version.

If the modules you need are already there, then great! You can skip this step.

If not, you'll need to set up a virtualenv. Check out these instructions for setting a virtualenv, and modify the steps as appropriate for your own dependencies.

When you've built your virtualenv, make a note of its path. They often live at /home/myusername/.virtualenvs/virtualenvname

/ > home > ykuo > myweddingstyle > code > webapp > my_app > static > css > beach.css *
